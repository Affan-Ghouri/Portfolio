# Affan Ghouri Portfolio

A modern, responsive portfolio website showcasing my work as a Software Engineer focused on Frontend Development and UI/UX Design.

## Tech Stack

| Technology | Version | Purpose |
|------------|---------|---------|
| React | 18.2.0 | UI Framework |
| Vite | 5.1.4 | Build Tool |
| Tailwind CSS | 3.4.1 | Styling |

## Features

- **Dark/Light Mode Toggle** - with neon theme
- **Loading Screen** - animated intro
- **Hero Section** - typing animation showcasing tech stack
- **About Section** - stats, focus areas, profile image
- **Skills** - progress bars with categories (Frontend, Tools, Design, AI, Other)
- **Projects** - filterable grid (Web App, UI/UX, Tool)
- **Contact** - form with email/WhatsApp, social links
- **Cursor Follower** - custom cursor effect
- **Scroll Reveal Animations**

## Custom Styling

- Neon glow effects (cyan #06b6d4)

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This project is configured with GitHub Actions for automatic deployment to GitHub Pages.

1. Push to `main` branch
2. Go to Repo → Settings → Pages
3. Select **GitHub Actions** as source
4. Wait for deployment to complete

Your site will be live at: `https://affan-ghouri.github.io/Portfolio/`
